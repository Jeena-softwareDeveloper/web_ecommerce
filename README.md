# Jeenora Wear - Premium E-commerce Platform

Jeenora Wear is a high-performance, multi-vendor e-commerce platform designed for the modern fashion industry. Built with a focus on speed, security, and a premium user experience.

## 🚀 Key Features

- **Multi-Vendor Ecosystem**: Specialized dashboards for suppliers and administrators.
- **AI-Powered Logistics**: Automated tracking and courier selection via Shiprocket.
- **Smart Notifications**: AI-drafted WhatsApp and email alerts for orders and status updates.
- **Secure Payments**: Integrated with Razorpay, Stripe, and Cashfree for seamless transactions.
- **Real-time Interaction**: Live chat support and order updates powered by Socket.io.
- **Production Hardened**: Built-in rate limiting, XSS protection, and secure session management.

## 🛠️ Technology Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Redux Toolkit, Framer Motion.
- **Backend**: Node.js, Express, MongoDB (Mongoose).
- **Services**: Cloudinary (Media), Shiprocket (Logistics), WhatsApp Web JS (Automation).

## 📦 Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables in `.env`
4. Start the development server:
   ```bash
   npm run start
   ```

## 🏗️ Build

To generate a production-ready bundle:
```bash
npm run build
```
